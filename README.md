## How to use

Make sure you have the following in your docusaurus config:

```ts
i18n: {
  defaultLocale: 'en',
  locales: ['en', 'fr'],
},
```

Then:

Run this command:

```bash
OPENAI_API_KEY=sk-xxxxxx npx docusaurus-i18n
```

## Get Help

```bash
npx docusaurus-i18n --help
```
